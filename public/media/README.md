# HORONHO Media Assets

Use this folder only for approved real assets.

## Folders

- `founder`: official founder portraits.
- `certificates`: scanned certificates or validated training documents.
- `gallery`: approved institutional photographs from Rabat, Casablanca, ministries, training sessions and missions.

## Rules

- Do not upload unapproved photographs.
- Do not publish certificates without checking names, dates and issuing institutions.
- Prefer optimized `.webp` or compressed `.jpg` for photos.
- Keep original high-resolution documents outside the public website when they contain sensitive data.
